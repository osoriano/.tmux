Install Guide
=============

1. Install or clone this directory into `~/.tmux`

2. Make a symlink from ~/.tmux/tmux.conf to ~/.tmux.conf (.i.e. run `ln -s ~/.tmux/tmux.conf ~/.tmux.conf`)

Enjoy!

Tmux Cheat Sheet
================
Copying text:
- C-[ to enter copy mode
- v to begin selection
- V to use line selection
- y to yank
- C-] to paste
