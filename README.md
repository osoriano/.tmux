Install Guide
=============

1. Clone this repo into `~/.tmux`

2. Create a symlink ot the repo (i.e. run `ln -s ~/.tmux/tmux.conf ~/.tmux.conf`)

Enjoy!

Tmux Cheat Sheet
================
Copying text:
- C-[ to enter copy mode
- v to begin selection
- V to use line selection
- y to yank
- C-] to paste
